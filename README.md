# ai-customer-service
AI智能客服系统（基于DeepSeek大模型的文档问答与知识库系统）

## 项目简介
本项目是基于大模型开发的企业级智能问答与知识库客服系统，支持文档上传、内容解析、知识库构建、智能问答、历史对话管理等功能。用户可向系统投喂PDF、TXT等文档，AI自动学习内容并实现精准问答，适用于客服辅助、内部知识库、智能咨询等场景，是典型的AI应用开发实战项目。

## 技术栈
- **前端**：Vue3 + JavaScript
- **后端**：Java + Spring Boot
- **AI模型**：DeepSeek 大模型 API
- **数据库**：MySQL
- **其他**：文件解析API、Postman（接口测试）、Git（版本控制）

## 核心功能
- **文档上传**：支持PDF、TXT等格式文档上传与解析
- **知识库构建**：自动提取文档内容，构建专属知识库
- **智能问答**：对接DeepSeek大模型，实现上下文理解、多轮对话、精准回答
- **对话管理**：存储用户对话记录，支持历史查询、关键词搜索
- **流式输出**：实现AI回复逐字输出，提升交互体验

## 项目职责（独立开发）
- 独立完成系统架构设计、前端界面开发、后端接口编写、大模型对接与调试
- 基于Vue3构建美观流畅的对话界面，实现消息流式输出、状态反馈等交互效果
- 完成文件上传、格式校验、内容解析、知识库管理模块开发
- 对接DeepSeek大模型API，实现智能问答、上下文理解、多轮对话能力
- 设计MySQL数据表存储用户信息、对话记录、文件数据
- 优化接口响应速度、加载状态、错误捕获与用户体验
- 掌握AI应用开发流程、Prompt工程、大模型调用与前后端数据交互
